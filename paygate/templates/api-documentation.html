SafePay's API is completely RESTful and all our responses are returned in JSON.

The SafePay API is deployed, so all you need to do is perform a HTTP request to the API endpoint.

  1. HTTP Method: POST

  2. Endpoint: /api

  3. Json Body contains these inputs: - Sender's Account Number, CVV, Expiry Date, Amount and Receiver's Account Number

Example Request:

curl --location --request POST 'http://127.0.0.1:8000/api'
--header 'Content-Type: application/json'
--data-raw '{
"sender_accno": 7731760391,
"cvv": 112,
"exp_date": "18/12/25",
"amount": 1,
"rec_accno": 2908566021
}'
